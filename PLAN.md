# LordEEN Tech — Privacy-Focused Lite Browser


## Overview
A sleek, dark-themed privacy browser called **LordEEN Tech** with built-in anti-detect tools, ad blocking, and an extensions/shields panel — all wrapped in a minimal, hacker-inspired design.

---

### **Features**
- **Browse the web** using an in-app browser with a URL/search bar
- **Incognito mode** toggle — clears cookies and history automatically when enabled
- **Built-in ad blocker** — injects scripts to block common ads and trackers
- **Extensions panel** with toggleable tools:
  - Ad Blocker (on/off)
  - Dark Mode Injector (forces dark CSS on websites)
  - Script Blocker (blocks JavaScript on pages)
  - Image Blocker (load pages faster without images)
- **Privacy shields panel** with toggleable protections:
  - Fingerprint Protection
  - Cookie Blocker
  - Tracker Blocker
  - WebRTC Leak Protection toggle
  - User Agent Switcher (appear as Chrome, Safari, Firefox, or custom)
- **Quick links** on the home screen for fast access to popular sites
- **Settings page** to manage default search engine and privacy defaults

---

### **Design**
- **Dark mode** throughout — true black (#000) backgrounds with dark gray (#1A1A1A) cards
- Subtle **cyan/teal accent** color for active states, toggles, and the logo
- Monospace-style typography for the URL bar giving it a techy feel
- Smooth toggle animations for shields and extensions
- Frosted glass-style bottom toolbar
- Minimal, distraction-free interface — content takes center stage

---

### **Pages / Screens**

1. **Home Screen**
   - Centered LordEEN Tech logo
   - Large search/URL bar below the logo
   - Row of quick-access site icons (Google, YouTube, GitHub, etc.)
   - Small incognito mode toggle in the corner

2. **Browser Screen**
   - Full-screen web view with a collapsible top URL bar
   - Bottom toolbar with back, forward, refresh, shields, and extensions buttons
   - Active shield/extension indicators (small badges)

3. **Extensions Panel** (slides up from bottom)
   - Grid of extension toggles with icons
   - Each extension has an on/off switch and brief description
   - Tools section: Ad Blocker, Dark Mode, Script Blocker, Image Blocker
   - Shields section: Fingerprint, Cookies, Trackers, WebRTC, User Agent

4. **Settings Screen**
   - Default search engine picker (Google, DuckDuckGo, Bing)
   - Privacy defaults (which shields are on by default)
   - Clear browsing data button
   - About section with app version
